* Para compilar con GCC: *

  $ gcc -g -Wall -o nombre nombre.c

-Wall: activa las advertencias
-o name: 'name' sera el nombre de salida
nombre.c: es el archivo con el codigo fuente

* Para ejecutar en UNIX like *

  $ ./nombre
